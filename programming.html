Notable Projects:

StreamCats.com

Description: A web tool for browsing the current Hulu and Netflix catalogues in a way that helps you discover new shows and movies fast.

Role/Scope: Full stack development - Front-end design, programming, and graphics. Backend API, database update pipeline, system administration.

2020 - Ongoing

DoubleDoppel

Description: Inspired by the impressive casting in the show 'Dark', where different versions of the same character span multiple timelines, DoubleDoppel is a client side machine learning application that uses facial recognition to compare a face in a reference image with 8000+ profiles on backstage.com and returns a list of actors/actresses with similar features.

Role/Scope: Data scraping and preprocessing, design, graphics, development and deployment.

2020

GetCoins ATM

Description: Developed UI software (Electron application) to run on Bitcoin ATM kiosks, and worked on the backend request network (gRPC and Protobufs with Golang) to handle transactions.

Role/Scope: Primarily backend development and system administration with some frontend UI work.

2019

DenverLightpost.com

Description: A location based digital flyer posting board for the Denver metro area. Users at specific "Lightposts" can post content and chat with other users within the Lightpost's radius.

Role/Scope: Full stack development - Frontend design, programming, and graphics. Backend API, System administration.

2018 - Ongoing

Parametric Designer / Programmer

Description: Python programmer and parametric design consultant for construction and fabrication projects ranging from custom CNC milling to hospitality, healthcare, and civic buildings.

Role/Scope: Used programming and digital modeling tools (Grasshopper 3d, Autodesk Revit, and Dynamo) to solve complex planning and fabrication problems during the value engineering and the final construction phases of building projects.
